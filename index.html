📋 GitHub 部署步驟
① 下載所有檔案，放入同一個資料夾
② 在 GitHub 建立新的 repository
③ git init → git add . → git commit -m "init PWA"
④ git push origin main
⑤ GitHub repo → Settings → Pages → Deploy from main 分支
⑥ 等待幾分鐘後訪問 https://<username>.github.io/<repo>/
⑦ 瀏覽器出現「安裝 App」提示 → 安裝到主畫面 ✓
